Tier 2

Galaxy Skeleton

bounded precompute · persisted artifact · schema v4

Region-cell lattice (spiral-ordered, circular footprint). Built once per galaxy, saved, loaded thereafter — hex generation only ever reads it.

Seeding passes (in order)

1 · Density summaryDONE

Per-cell mean density, void flag, connectivity graph; chokepoints = articulation cells.

2 · Stellar lean & metallicityDONE

Young-bright / old-dim / remnant-graveyard leans + metallicity — species terrain and hex star-table bias.

3 · Resource anchorsDONE

Mineral-rich systems, precursor sites as anchored hexes (one anchor per hex, collision-probed).

4 · Homeworlds & speciesDONE

Spacing-aware sapient origins; species profiles — embodiment + temperament axes (6 scalars).
